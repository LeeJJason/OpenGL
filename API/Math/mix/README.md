## Declaration

```c
genType mix(	genType x,
 	genType y,
 	genType a);
 
genType mix(	genType x,
 	genType y,
 	float a);
 
genDType mix(	genDType x,
 	genDType y,
 	genDType a);
 
genDType mix(	genDType x,
 	genDType y,
 	double a);
 
genType mix(	genType x,
 	genType y,
 	genBType a);
 
genDType mix(	genDType x,
 	genDType y,
 	genBType a);
 
genIType mix(	genIType x,
 	genIType y,
 	genBType a);
 
genUType mix(	genUType x,
 	genUType y,
 	genBType a);
 
genBType mix(	genBType x,
 	genBType y,
 	genBType a);
 
```

## Parameters

- **`x`**

  指定用于插值的范围开始

- **`y`**

  指定用于插值的范围结束

- **`a`**

  指定用于在*`x`* 和*`y`*之间插值的值
